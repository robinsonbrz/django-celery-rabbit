## Executando e debugando o container Docker localmente


Criar um arquivo **.env** baseado no exemplo **.envEXAMPLE**

Remover builds anteriores (opcional)

```
docker-compose down
```

Build dos containers

```
docker-compose -f docker-compose-dev.yml up -d --build
```

Esse comando build cria uma imagem Docker, e dá um "start" em um container que utilizaremos para debug e desenvolvimento.

Ao finalizar o build, a aplicação estará disponível em localhost:<porta-defeinida-em-.env>
